# FastDFS 重大版本一览

* V1.0： 2008年 7月 第一个版本，采用一个请求一个线程模型
* V2.0： 2010年 8月 使用libevent支持异步网络IO，支持100K级连接
* V3.0： 2011年 6月 采用小文件合并存储，支持海量小文件
* V4.0： 2012年10月 支持storage server id特性，运维更友好
* V5.0： 2013年12月 移除libevent，原生实现网络异步IO
* V6.0： 2019年10月 支持双IP，更好支持NAT网络和跨机房部署
* V6.09：2022年 9月 依赖libserverframe网络基础库
* V6.11：2023年12月 支持IPv6，支持IPv4和IPv6双栈
* V6.13：2025年 8月 支持读写分离，可实现跨机房灾备
* V6.15：2025年10月 支持多线程文件同步，加快同步速度
* V6.16：2026年 6月 支持一台服务器部署多个同组的storage实例
