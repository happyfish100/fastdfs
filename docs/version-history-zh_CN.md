# FastDFS 重大版本一览

* V1.0 ~ V1.29：2008年 7月至2010年 6月 采用一个请求一个线程模型
* V2.0 ~ V2.09：2010年 8月至2011年 2月 使用libevent支持异步网络IO，支持100K级连接
* V3.0 ~ V3.11：2011年 6月至2012年 8月 实现小文件合并存储，支持海量小文件
* V4.0 ~ V4.08：2012年10月至2013年11月 支持storage server id特性，运维更友好
* V5.0 ~ V5.12：2013年12月至2018年 6月 移除libevent，原生实现网络异步IO
* V6.0 ~ V6.16：2019年10月至2026年 6月 支持双IP，更好支持NAT网络和跨机房部署
* V6.09：2022年 9月 依赖libserverframe网络基础库，修复了V2.0引入在特定情况下CPU跑满的问题
* V6.11：2023年12月 支持IPv6，实现IPv4和IPv6双栈
* V6.13：2025年 8月 支持读写分离，可实现跨机房灾备
* V6.15：2025年10月 支持多线程文件同步，加快同步速度
* V6.16：2026年 6月 支持一台服务器部署同组的多个storage实例
